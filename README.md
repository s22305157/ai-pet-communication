# PAWLINK - AI 寵物溝通師 (AI Pet Communicator)

PAWLINK 是一個專為寵物愛好者設計的 AI 溝通應用程式，利用先進的 AI 技術協助主人更好地理解毛小孩的需求與情感。

## 專案目標

- **即時溝通**：利用 AI 模擬與毛小孩的對話。
- **寵物管理**：集中管理多位毛小孩的個人檔案。
- **專業體驗**：提供會員制度與精美的 UI 介面，讓使用者享受頂級服務。

## 技術棧

- **框架**：Flutter (Android, iOS, Web)
- **後端**：Firebase (Authentication, Firestore, Storage)
- **本地存儲**：Hive
- **設計**：Google Fonts, Animations

## 開始使用

1. **環境設定**：確保已安裝 Flutter SDK。
2. **安裝依賴**：
   ```bash
   flutter pub get
   ```
3. **執行專案**：
   ```bash
   flutter run
   ```
